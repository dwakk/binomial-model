# Binomial Options Pricing Model

A Python visualizer for the Binomial Options Pricing Model with Black-Scholes convergence analysis.

## Features
- Interactive binomial tree with zoom/pan
- Real-time parameters adjustments
- Payoff vs profit display toggle
- Most likely path visualization
- Convergence analysis vs Black-Scholes
- Export convergence data to CSV

## Quick Start
```bash
git clone https://github.com/dwakk/binomial-model.git
cd binomial-model
pip install -r requirements.txt
python main.py
```
